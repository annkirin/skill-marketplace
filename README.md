# Skill Marketplace

通用 AI 技能集市，为 AI 编码助手提供可复用的技能包。

## Skills

| Skill | Description |
|-------|-------------|
| [brainstorming](skill/brainstorming/SKILL.md) | 通用头脑风暴流程，任务开始前使用 |
| [automated-testing](skill/automated-testing/SKILL.md) | 自动化测试 + 模拟人类实操验证，代码完成后使用 |
